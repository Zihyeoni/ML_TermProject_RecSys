# TasteMate
개인 맞춤 음식 추천 시스템
  
팀원 | 이현지, 팜티투짱, 박성언, 김연주

### **Filtering Used**  
* Collaborative filtering (model-based)
* Content-based filtering

| 브랜치           | 작업 내용                   |
| --------------- | ----------------------- |
| `preproc`     | 데이터 전처리             |
| `model-cf`      | Collaborative Filtering |
| `model-cbf`     | Content-based Filtering |
| `visualization` | 평가 결과 및 시각화             |
| `main`          | 병합 및 정리             |
